Project 1

This script is to help with collecting all the log files from a computer and moving them to a central repository. This be scaled out to utilize task schedular or active directory to collect all the files from all comptuers that are managed.

The Folder is named for the computer for easier searchability and the logs have been placed inside a log folder to allow this script to expaned if other files are needing to be transfered in the process.

You will need to change the destination location, whether it is a network location or a usb drive.

Run the application as an administrator for sucessful completion.
